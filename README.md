# inv360

A simple inventory management system. (under development)

## Technologies

- Java 25
- Spring Boot
- Gradle
- Spring Data JPA
- PostgreSQL
- Redis

## Features

- product management
- warehouse management
- inventory management
- stock movement tracking
- basic exception handling

## Running

Run the application:

```bash
./gradlew bootRun
```

On Windows:

```bash
gradlew.bat bootRun
```

## Tests

```bash
./gradlew test
```
